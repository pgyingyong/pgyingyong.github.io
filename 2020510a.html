记一次整同事的经历

我在公司有个同事，搞C的，好欠揍。有一天我心生一计……
我把他的C语言改了下，他现在怎么都改不了代码了，现在教给你们：
1，多用函数，把他搞晕
源代码：printf("hello");
修改后：
void aa()
{
ac();
}
void ab()
{
printf("hello");
}
void ac()
{
}
int main()
{
aa();
return 0;
}
还可以再多点，
嘿嘿
2，变量名称简简单
源代码：int gold=10;
修改后：int a=10;
3，赋值多用计算
源代码：int a=10;
修改后：int a=3*3+3*0.33333+0.11111-0.1111;
嘿嘿
4，不写注释
源代码：int a=3*3+3*0.33333+0.11111-0.1111; //等于10
修改后（不用说了）：int a=3*3+3*0.33333+0.11111-0.1111;
好像废话
5，多用全局变量
源代码：int main()
{
int *p = 10;
return 0;
}
修改后：
int p=10;
int main()
{
return 0;
}
拿全局变量，让他翻代码翻累，嘿嘿
6，拼音注释（你想写注释就执行这条）
源代码：printf("hi");
//Output Hi
修改后：printf("hi");
//sc hi
最狠的是只用拼音首字母……
以上就是如何整同事的六个技巧的全部内容了，拜拜！
